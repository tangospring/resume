1. You will need a server to host our app
    • I recommend downloading Netbeans for 2 reasons:
      1. it will run most applications that we have on Code For America-KC
      2. it integrates well with Git.(great IDE for beginners)


  2. You need your own copy("Fork") of this project
    • Make your own Fork of the app in your Github account or by cloning the project on your machine.


  3. Clone your project into Netbeans
    1. On the toolbar click Team/Git/Clone...
      1. Enter this Repository URL: https://github.com/codeforkansascity/Code-for-America-Projects-Hub.git
        leave the User and Password fields blank, click next.
      2. Ensure that the gh-pages* branch is selected, click next.
      3. Leave these fields as they are.
        Ensure that Scan for Netbeans Projects after Clone is checked, click finish.


  4. Import your project into Netbeans
    1. click Create Project...
    2. click HTML5 Application with Existing Sources, click next.
    3. Next to Site Root click Browse... to navigate to the project you just cloned.
    4. Click Code-for-America-Projects-Hub folder, click Open.
    5. Step 4 should auto-populate all fields, leave them as they are, click Finish.
    6. You are done importing the project. you should see it appear in the Projects sidebar.
    7. click Run/Select Project Browser.
      select the browser you would like to use. (dont select the embedded browser or one with a netbeans connector)
    8. in the main project directory, right-click on index.html and select Run File
    9. You should see the main project open in your browser.


  5. Making your first changes and submitting("Push") them
    1. in the main project directory double-click new_members_test.html to open for editing
    2. click HTML5 Application with Existing Sources, click next.
    3. Next to Site Root click Browse... to navigate to the project you just cloned.
    4. Click Code-for-America-Projects-Hub folder, click Open.
    5. Step 4 should auto-populate all fields, leave them as they are, click Finish.
    6. You are done importing the project. you should see it appear in the Projects sidebar.
